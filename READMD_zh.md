# DayZ Mod Setup Tool

此工具能够创建新服务器时挂载MDO或者给现有的服务器进行新MOD的挂载


## 环境依赖

在运行此项目之前，请确保系统安装了以下依赖：

- [Node.js](https://nodejs.org/) (版本 = 16.18.1)



## 安装步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/yowainaegi/dayz-mod-setup-tool.git
   cd dayz-mod-setup-tool
   ```

2. 安装依赖：

   ```bash
   npm install
   ```

3. 运行：

   ```bash
   npm run electron:serve
   ```

4. 然后即可使用
